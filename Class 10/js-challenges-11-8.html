    Challenge 5

    The following form takes in 3 inputs: width, length, and height. After a user clicks "Submit Dimensions", the form should tell the user the volume of their box. Also, if the user types in a blank value or a non-number, an error should appear on the page to let them know. It might tell the user "Please submit valid values" or something similar. You will need to get the length, width, and height values stored in variables. After this, you'll create a variable to store their total when multiplied together (volume). You can then use "isNaN" to check if the volume is a number or not. If it is, output the information. Otherwise, you would tell the user to put in valid values.
