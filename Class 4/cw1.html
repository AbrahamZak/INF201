Practice Validation Assignment

Your job is to get this HTML document to validate. In order to do so, you must to go the W3 Validator, paste your code, and then hit check. If you see only a green bar, then you have completed the assignment. A red/yellow bar indicates that work still needs to be done before the page meets good validation standards.

There are numerous errors on this page and they can appear anywhere in the code. If you right-click the page in Google Chrome and select View page source, the HTML for this page will appear. It is important to note that no content should be deleted. Tags should only be added, moved around, or deleted in order to pass validation. Don't worry about styles as the primary focus of this assignment should be on code validation.

To submit the assignment, you will upload the corrected file to your S-Drive via FileZilla or on a UAlbany computer. The link should look like this: albany.edu/~username/inf362/2-6/homework.html. Username will be replaced with your personal UAlbany ID.
